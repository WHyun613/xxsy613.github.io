计算机网络

RTT Round Trip Time 发送数据到接受到接收端的确认，经历的时延

OSI Open System Interconnection Model 计算机网络体系结构，物理层，数据链路层，网络层，传输层，会话层，表示层，应用层

TCP/IP Protocol Suite 计算机网路体系结构，网络接口层，网际层，传输层，应用层

SAP Service Access Point 服务访问点，同一系统中相邻两层的实体交换信息的逻辑接口，用于区分不同的服务类型，表示上层为下层提供服务的点

PDU Protocol Data Unit 协议数据单元，对等层次之间传送的数据包称为该层的协议数据单元（帧、分组、报文）PDU = PCI + SDU

SDU Segment Data Unit 服务数据单元，同一系统内，层与层之间交换的数据包称为服务数据单元

PCI Protocol Control Information 协议控制信息，控制协议操作的信息

RZ Return To Zero 归零编码把时钟信号用归零的方式编码在了数据内，称为“自同步”信号

NRZ Non Return To Zero 非归零编码，需要时钟信号线，即外同步（以下四种都可以自同步）

NRZI Non Return To Zero Reversal 反向非归零编码，既能传输时钟信号，又不损失系统带宽

ASK Amplitude Shift Keying 幅移键控，通过改变载波信号的振幅来表示数字信号1和0，载波的频率和相位不发生改变

**FSK Frequency Shift Keying **频移键控，通过改变载波信号的频率来表示数字信号1和0，载波的振幅和相位不发生改变

PSK Phase Shift Keying 相移键控 ，通过改变载波信号的相位来表示数字信号1和0，载波的振幅和频率不发生改变，又分为绝对调相和相对调相

QAM Quadracture Amplitude Modulation 正交振幅调制 ，频率相同下，将ASK和PSK结合，形成叠加信号

MTU Maximum Transmission Unit 最大传输单元

**CRC Cyclic Redundancy Code **循环冗余码

FCS Frame Check (Checking) Sequence 帧检验序列

ARQ Automatic Repeat reQuest 自动重传请求，通过接收方请求发送方重传出错的数据来回复出错的帧，是通信中用于处理信道所带来查错的方法之一

SW Stay-and-Wait 停止等待协议，流量控制的方法，每次只允许发送一帧，然后等待接收方的确认帧

GBN Go-Back-N 后退N帧协议，可以连续发送帧，但是接收方只允许按顺序接受帧

SR Selective Repeat 选择重传协议，设法只重传出现差错的数据帧，发送窗口和接收窗口为$2^{n-1}$个

FDM Frequency Division Mutiplexing 频分多路复用，将多路基带信号调制到不同频率载波上，再叠加形成一个复合信号(相邻信道需要隔离频道，防止干扰)

TDM Time Division Mutiplexing 时分多路复用，是将一条物理信道按时间分成若干时间片，轮流地分配给多个信号使用

WDM Wave Division Multiplexing 波分多路复用 在一根光纤中传输多种不同波长(频率)的光信号，由于波长(频率)不同，各路光信号互不干扰，最后再用波长分解复用器将各路波长分解出来

CDM(CDMA) Code Division Multiple Access 码分多路复用，每一个用户可以在同样的时间使用同样的频带进行通信，共享时间空间

ALOHA Addictive Link On-line HAwaii system 分为纯ALOHA和时隙ALOHA，前者是自由发送数据帧，后者只能在每个时隙开始时发送帧，不听就发

CSMA Carrier Sence Multiple Access 载波监听多路访问，分为1-坚持CSMA，非坚持CSMA，p-坚持CSMA，先听再发

CSMA/CD Carrier Sence Multiple Access/Collision Detection 载波监听多路访问/碰撞检测，应用于有线连接的局域网，先听再发，边听边发，冲突停发，随机重发

CSMA/CA Carrier Sence Multiple Access/Collision Avoidence 载波监听多路访问/碰撞避免，应用于无线连接的局域网

IFS Inter Frame Space 帧间间隔，所有的站点必须在持续检测到信道空闲一段指定时间后才能发送帧

SIFS Short Inter Frame Space 短帧间间隔，短

PIFS Point Inter Frame Space 点协调间隔，中

DIFS Distributed Inter Frame Space 分布式协调间隔，长

RTS Request To Send 请求发送控制帧

CTS Clear To Send 允许发送控制帧

LAN Local Area Network 局域网

VLAN Virtual Local Area Network 虚拟局域网

LLC Logic Link Control Layer 逻辑链路控制子层，向网络层提供无确认无连接、面向连接、带确认无连接、高速传送4种不同的连接服务类型

MAC Medium Access Control 介质访问控制子层，向上层屏蔽对物理层访问的各种差异，提供对物理层的统一访问接又，主要功能包括:组帧和拆卸帧、比特传输差错控制、透明传输，用来决定广播信道中信道分配的协议属于数据链路层的一个子层

MAC Medium Access Control 介质访问控制地址，MAC地址是对网络上各接又的唯一标识，而不是对网络上各设备的唯一标识

PPP Point to Point Protocol 点对点连接协议，广域网数据链路层协议

LCP Link Control Protocol 链路控制协议 LCP，用于建立、配置以及测试数据链路的连接(物理)

NCP Network Control Protocol 一套网络控制协议NCPs其中的每一个协议支持不同的网络层协议(逻辑)

SDN Software Define Network 软件定义网络，南向接口北向接口

IPV4 Internet Protocol Version 4

IPV6 Internet Protocol Version 6

MF More Fragment 为1时表示后面还有分片

DF Don’t Fragment 为1时不允许分片，为0时允许分片

NAT Network Address Translation 网络地址转换

CIDR Classless Inter-Domain Routing 无分类域间路由选择，IP地址的划分方法

ARP Address Resolution Protocol 地址解析协议，完成IP地址到MAC地址的映射，工作在网络层

DHCP Dynamic Host Configuration Protocol 动态主机配置协议，用于给主机动态的分配IP地址，应用层协议，基于UDP

ICMP Internet Control Message Protocol 网际控制报文协议，让主机或路由器报告差错和异常情况，给IP数据报加上一个ICMP首部变为ICMP报文，网络层协议

PING Packet Internet Groper 分组网间探测，ICMP的应用，用来测试两主机之间的连通性，工作在应用层

**AS Autonomous System **自治系统，单一技术管理下的路由器合集

IGP Interior Gateway Protocol 内部网关协议，在自治系统内部使用的路由选择协议

​ RIP Routing Information Protocol 路由选择信息协议，IGP协议，分布式的基于距离向量的路由选择协议，应用层协议基于UDP

​ OSPF Open Shortest Path First 开放最短路径协议，IGP协议，分布式链路状态路由算法，网络层协议基于IP

EGP External Gateway Protocol 外部网关协议，在两个不同的自治系统之间使用的路由选择协议

​ BGP Border Gateway Protocol 边界网关协议，EGP协议，是不同自治系统的路由器之间交换信息的协议，应用层协议基于TCP

IGMP Internet Group Management Protocol 因特网组管理协议

UDP User Datagram Protocol UDP数据报协议，传输层协议，提供复用，分用和差错检测，无连接不可靠

TCP Transmission Control Protocol 传输控制协议，提供有连接可靠的服务，让传输可靠有序无丢失不重复

URG Urgent 紧急位，TCP数据报的标识位，当值为1时表示该数据报为紧急数据，设为最高优先级

ACK ACKnowledgment 确认位，当值为1时确认号字段才有效

PSH Push 推送位，接受方收到值为1的报文段，应尽快交付给应用进程，与URG效果相同

RST Reset 复位位，当值为1时，表示TCP连接出现严重差错，必须释放连接

SYN Synchronous 同步位，当值为1时，表示这是一个连接请求或连接接受报文

FIN Finish 终止位，当值为1时，表示发送方数据发送完毕，要求释放连接

rwnd Receive Window 接收窗口，负责点到点的流量控制

cwnd Crowd Window 拥塞窗口，负责全局性的拥塞控制

C/S Client/Service 客户服务器模型，应用层

P2P Peer To Peer 点对点模型，应用层

DNS Domain Name System 域名系统，把网址转换为IP地址，使用无连接的UDP数据报，所属应用层

FTP File Transfer Protocol 文件传输协议，使用TCP的可靠服务

UA User Agent 用户代理，电子邮件的组成结构

SMTP Simple Mail Transfer Protocol 邮件发送协议，用于用户代理向邮件服务器发送邮件或在邮件服务器之间发送邮件

POP3 Post Office Protocol - Version 3 邮件读取协议，用于用户代理从邮件服务器读取邮件，只下载邮件，不做同步

IMAP Internet Message Access Protocol 因特网消息访问协议，跟POP3相同作用，下载并同步所有信息，两台电脑上的邮件系统会完全相同

MIME Multipurpose Internet Mail Extensions 多用途网络邮件扩充，定义了传送非ASCII码的规则，优化了SMTP

WWW World Wide Web 万维网，一个分布式，联机式的信息存储空间

URL Uniform Resource Locator 统一资源定位符，使每一资源在万维网范围内有唯一标识符

HTTP **HyperText Transfer Protocol **超文本传输协议，应用层协议，使用TCP，CS

HTML HyperText Markup Language 超文本标记语言，前端语言

操作系统

先来先服务 first-come first-serverd（FCFS）

最短作业优先 shortest job first（SJF）

最佳（OPT）置换算法：从主存中移出永远不再需要的页面；如无这样的页面存在，则选择最长时间不需要访问的页面。于所选择的被淘汰页面将是以后永不使用的，或者是在最长时间内不再被访问的页面，这样可以保证获得最低的缺页率。 即被淘汰页面是以后永不使用或最长时间内不再访问的页面。

先进先出（FIFO）置换算法：是最简单的页面置换算法。这种算法的基本思想是：当需要淘汰一个页面时，总是选择驻留主存时间最长的页面进行淘汰，即先进入主存的页面先淘汰。其理由是：最早调入主存的页面不再被使用的可能性最大。 即优先淘汰最早进入内存的页面。（往前看）

最近最久未使用（LRU）算法：这种算法的基本思想是：利用局部性原理，根据一个作业在执行过程中过去的页面访问历史来推测未来的行为。它认为过去一段时间里不曾被访问过的页面，在最近的将来可能也不会再被访问。所以，这种算法的实质是：当需要淘汰一个页面时，总是选择在最近一段时间内最久不用的页面予以淘汰。 即淘汰最近最长时间未访问过的页面。（往前看）
